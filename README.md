# Travel Speed Time Calculator

A beautiful, modern web application that calculates the time saved when increasing your travel speed.

## Features

- **Time Saved Calculation**: Instantly see how many minutes you save.
- **Percentage Analysis**: Understand the relative benefit of speeding up.
- **Modern UI**: Clean, responsive design with smooth animations.
- **Instant Feedback**: Real-time results without page reloads.

## Prerequisites

- Python 3.x
- pip (Python package installer)

## Installation

1.  Clone the repository (or download the files).
2.  Navigate to the project directory:
    ```bash
    cd travel_speed_time_calculator
    ```
3.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Start the application:
    ```bash
    python app.py
    ```
2.  Open your web browser and go to:
    `http://127.0.0.1:5000`

## Testing

To run the unit tests:

```bash
pytest
```

## Project Structure

- `app.py`: Main Flask application backend.
- `templates/index.html`: Frontend HTML structure.
- `static/css/style.css`: Custom styling.
- `static/js/script.js`: Frontend logic and API interaction.
- `tests/`: Unit tests.
